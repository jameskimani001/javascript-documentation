Introduction to JavaScript

JavaScript is a versatile programming language primarily used for web development.

It allows you to create interactive websites and applications.

JavaScript can be embedded directly in HTML pages.

Example of a simple JavaScript alert:

alert('Hello World');
  • Client-side scripting
  • Event handling
Variables and Types

JavaScript variables can hold different data types such as numbers, strings, and objects.

Use var, let, or const to declare variables.

Example:

let name = 'Alice'; const PI = 3.14;
  • String
  • Number
  • Boolean
Functions

Functions are reusable blocks of code designed to perform a specific task.

Functions can take parameters and return values.

Example function:

function greet(name) {
  return 'Hello, ' + name;
}
  • Function declaration
  • Function expression
Control Flow

Control flow statements determine the order in which code executes.

Common statements include if, else, and switch.

Example:

if (age >= 18) { console.log('Adult'); }
  • Conditional statements
  • Loops (for, while)
DOM Manipulation

The Document Object Model (DOM) represents the structure of an HTML document.

JavaScript can manipulate DOM elements to update content dynamically.

Example:

document.getElementById('title').textContent = 'New Title';
  • Access elements
  • Modify content and styles
